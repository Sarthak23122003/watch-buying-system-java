# ⌚ Watch Buying System (Java)

A console-based Java application that simulates an online watch shopping system.

## 🚀 Features
- Add watches to cart
- View cart items
- Checkout with total price
- Uses OOP concepts (Inheritance, Interface, Classes)

## 🛠️ Technologies Used
- Java
- OOP (Object-Oriented Programming)

## 📂 Project Structure
- WatchBuying.java

## ▶️ How to Run
1. Compile:
   javac WatchBuying.java

2. Run:
   java WatchBuying

## 📸 Sample Output
User can:
- Add watches
- View cart
- Checkout
